Introdução Seletores Box Model Layout Moderno

1. Introdução ao CSS

O CSS controla como os elementos HTML são exibidos na tela. Ele lida com o "design", permitindo que você altere cores, fontes, e a posição dos elementos.

Cascata: O nome "Cascading" significa que se houver duas regras conflitantes, a que tiver maior especificidade ou a que aparecer por último no código será a aplicada.

2. Seletores e Sintaxe

Para aplicar estilo, você precisa "apontar" para o elemento HTML:

h1 {
  color: red;
  font-size: 32px;
}
  • Seletor de Elemento: p { } (afeta todos os parágrafos)
  • Seletor de Classe: .minha-classe { } (usa o atributo class="minha-classe")
  • Seletor de ID: #meu-id { } (único para um elemento específico)

3. O Modelo de Caixa (Box Model)

Todo elemento no CSS é visto como uma caixa retangular. Ele é composto por:

  • Content: O conteúdo real (texto/imagem).
  • Padding: Espaço interno entre o conteúdo e a borda.
  • Border: A borda que envolve o padding e o conteúdo.
  • Margin: Espaço externo, fora da borda, para separar de outros elementos.
Exemplo visual de um elemento com Padding e Border

4. Layout Moderno (Flexbox & Grid)

Antigamente usávamos "floats" para organizar sites. Hoje, usamos ferramentas mais poderosas:

Flexbox

Ideal para alinhar elementos em uma única dimensão (linha ou coluna).

.container {
  display: flex;
  justify-content: space-around;
}

CSS Grid

Perfeito para layouts complexos em duas dimensões (linhas e colunas simultâneas).
